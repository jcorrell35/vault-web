# Quick Start Key Binds for VIM

| Key Bind | Command Action                                   |
| -------- | ------------------------------------------------ |
| **h**    | left                                             |
| **j**    | down                                             |
| **k**    | up                                               |
| **l**    | right                                            |
| **r**    | replace                                          |
| **d**    | delete                                           |
| **dd**   | delete line                                      |
| **dw**   | delete word                                      |
| **d$**   | delete from character to end of line             |
| **d_**   | delete from character to first character on line |
